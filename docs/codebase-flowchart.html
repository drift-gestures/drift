1. Active runtime path

  • Copies stable contact fields

C to Swift boundary

Own the snapshot

CTrackpadBridge.swift

Immediately copies the borrowed C pointer into Swift FingerContact and TrackpadSnapshot values.

→
Swift orchestration

Distribute one frame

SwiftBridge.swift

Processes listeners synchronously under a lock, applies suppression, then delivers logging and frontend updates on the main actor.

SwiftBridge fans the copied snapshot into four responsibilities
↓
Recognition

ListenerPipeline.swift

Calls listeners in registration order and arbitrates stop-propagation and interaction claims.

Current output: empty because AppDelegate passes listeners: [].
OS interference

EventSuppressionController.swift

Owns the CGEvent tap. It can remove one requested scroll axis/direction or consume press events.

Current output: no suppression requests.
Testing state

ActivityLogStore.swift

Keeps up to 300 entries and samples raw frame, phase, contact count, center, scale, rotation, and listener transitions.

Rendered by LoggingView.swift.
Frontend state

HUDStore.swift

Stores active HUD IDs, optional custom state, and the latest TrackpadState for future HUD definitions.

Current output: no active HUDs.
