# Drift user guide

Drift is a macOS menu-bar app for trackpad gestures, floating HUDs, and custom actions. Open **Settings** from Drift's menu-bar menu to configure its features.

## Guides

- [Custom gestures](custom-gestures.md) — create trackpad gestures and assign keyboard-shortcut sequences or other actions.

## Before you begin

Drift needs **Input Monitoring** and **Accessibility** permissions in macOS System Settings → Privacy & Security to observe gestures and send keyboard shortcuts. If **Settings → General** shows Input Suppression as **Disabled** after granting permission, select **Retry**.
