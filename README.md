# drift

drift is a macOS menu-bar utility for recognizing custom trackpad gestures and performing user-configured actions.

## Build

Open `drift.xcodeproj`, select the `drift` scheme and **My Mac**, then Run. No Node/npm tooling is required.

## Preserved drawings

Previous Excalidraw drawing files are not modified or deleted by this release. To continue using a drawing, upload its `.excalidraw` file manually at [excalidraw.com](https://excalidraw.com).
